# 企融智配 - 企业融资智能匹配助手

## 项目简介
基于企业融资需求的智能产品匹配工具，帮助四川地区企业快速找到合适的银行融资产品。

## 部署方式

### Vercel 部署（推荐）

1. 将代码推送到 GitHub
2. 登录 [vercel.com](https://vercel.com)，用 GitHub 账号登录
3. 点击 "Add New Project"
4. 选择你的 GitHub 仓库
5. 直接点击 Deploy，无需修改配置
6. 等待 1-2 分钟，获得 `https://xxx.vercel.app` 链接

## 功能特点
- 5维度智能匹配：成立年限、纳税等级、年纳税额、年营收、抵押物
- 45+ 银行产品知识库
- 手机号留资功能
- 企业微信二维码预约咨询

## 技术栈
- 纯 HTML + CSS + JavaScript
- 本地数据存储（localStorage）
- 静态部署
